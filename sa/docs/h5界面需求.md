# 文献辅助阅读系统 — H5移动端界面与核心功能需求

## 1. 页面结构与模块

- 登录/注册页面
  - 与Web端账号体系共用

- 首页
  - 分类（文件夹）浏览，树状/列表结构；切换类目，展示文献列表

- 文献详情页
  - 文献信息（标题、作者、摘要、上传时间），PDF预览页（支持文本摘要/简要预览）
  - 最近笔记、历史AI问答区
  - 支持为选中段落做AI提问或记笔记

- AI对话页
  - 针对当前文献或全局跨文献提问
  - 展示历史问答记录

- 笔记管理页
  - 笔记编辑、查看、同步状态提示

## 2. 核心功能

- 用户登录/注册/同步
  - 和Web同步所有数据（类目、文献、笔记、AI记录）
- 文献与类目浏览
  - 导航文件夹，选择、预览、查询文献
  - 自动同步后端结构与阅读进度
- 文献阅读与交互
  - PDF页面（可根据设备适配只看某页或只看重点摘要）
  - 支持标记重要片段及备注
  - 支持弹窗快捷操作：AI问答、做笔记
- AI交互
  - 跨文献提问（如：查某关键词出现在哪些文献/笔记）
  - 最近AI对话列表
- 笔记功能
  - 新建、编辑、保存、同步、状态反馈

## 3. 非功能需求

- 兼容性：适配主流移动浏览器（微信、Safari移动端等）
- 安全性：全部接口需JWT校验
- 稳定性：所有数据同步失败有提示，AI接口异常记录日志

## 4. 技术建议

- 框架：Vue3 + Vant
- 移动端文件浏览：优化性能与展示（虚拟列表）
- 状态同步：REST接口+WebSocket（笔记/AI实时同步）

## 5. 原型交互重点

- 首页分类-文献-笔记-详情层次导航完整串联
- PDF预览与重点标记交互
- AI对话/笔记页快速切换与上下文提示
