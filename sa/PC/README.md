# 文献辅助阅读系统 (Literature Assisted Reading System)

## 项目简介

文献辅助阅读系统是一个专为学术研究人员、学生和知识工作者设计的智能文献阅读平台。系统集成了PDF阅读、AI问答、智能笔记、分类管理等功能，旨在提升文献阅读效率和知识管理能力。

## 核心特性

### 📚 智能PDF阅读
- 高性能PDF渲染引擎
- 多种阅读模式（单页、连续、对页）
- 自适应缩放和布局
- 阅读进度自动保存
- 文本选择和复制

### 🤖 AI智能问答
- 基于文档内容的智能问答
- 支持多种AI模型（GPT-4、Claude等）
- 上下文记忆和对话历史
- 智能问题推荐
- 引用来源标注

### 📝 智能笔记系统
- 文档关联笔记
- 富文本编辑器
- 标签分类管理
- 快速检索和筛选
- 笔记导出功能

### 🗂️ 分类管理
- 多级分类树结构
- 拖拽式分类操作
- 批量文档管理
- 分类统计分析
- 自定义分类颜色

### 🔍 全文搜索
- 文档内容全文检索
- 笔记和对话搜索
- 高级筛选条件
- 搜索结果高亮
- 智能搜索建议

### ☁️ 数据管理
- 本地数据存储
- 自动备份机制
- 数据导入导出
- 云端同步支持
- 隐私保护

## 快速开始

### 环境要求
- 现代Web浏览器（Chrome 90+, Firefox 88+, Safari 14+, Edge 90+）
- 本地Web服务器（推荐使用Live Server或类似工具）

### 安装步骤

1. **克隆或下载项目**
   ```bash
   # 如果使用Git
   git clone <repository-url>
   cd PC
   ```

2. **启动本地服务器**
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 或使用Node.js
   npx serve .
   
   # 或使用PHP
   php -S localhost:8000
   ```

3. **访问系统**
   打开浏览器访问 `http://localhost:8000`

## 功能使用指南

### 1. 登录系统
- 使用默认账户或注册新账户
- 支持"记住我"功能
- 忘记密码可通过邮箱重置

### 2. 文档管理
- **上传文档**: 在仪表板点击"上传PDF"按钮
- **分类管理**: 使用左侧分类树进行文档分类
- **搜索文档**: 使用顶部搜索框快速查找

### 3. PDF阅读
- **打开文档**: 在文档列表中点击文档名称
- **阅读控制**: 使用工具栏进行缩放、翻页等操作
- **文本选择**: 选中文本后可创建笔记或提问AI

### 4. AI问答
- **提问方式**: 选中文本后点击"询问AI"或直接在右侧面板输入问题
- **查看历史**: 在AI对话历史页面查看所有对话记录
- **导出对话**: 支持多种格式导出对话内容

### 5.笔记管理
- **创建笔记**: 选中文本后点击"做笔记"或在笔记页面新建
- **编辑笔记**: 支持富文本编辑和标签管理
- **搜索笔记**: 按文档、标签、时间等条件筛选

### 6. 个人设置
- **个人信息**: 修改用户名、邮箱等基本信息
- **阅读偏好**: 自定义PDF阅读体验
- **AI设置**: 配置AI模型和回答风格
- **系统设置**: 界面语言、通知等系统配置

## 性能优化建议

### 1. 文档管理
- 单个PDF文件建议不超过50MB
- 同时打开的文档数量建议不超过5个
- 定期清理不需要的文档和笔记

### 2. 浏览器设置
- 启用硬件加速
- 分配足够的内存给浏览器
- 定期清理浏览器缓存

### 3. 网络优化
- 使用稳定的网络连接
- AI功能需要良好的网络环境
- 考虑使用CDN加速静态资源

## 常见问题

### Q: PDF文档无法正常显示？
A: 请检查文档是否损坏，浏览器是否支持PDF.js，或尝试刷新页面。

### Q: AI问答功能无响应？
A: 请检查网络连接，确认AI服务配置正确，或尝试切换AI模型。

### Q: 数据丢失怎么办？
A: 系统使用本地存储，建议定期导出数据备份，启用自动备份功能。

### Q: 如何提高阅读性能？
A: 关闭不必要的浏览器标签页，调整PDF显示质量，使用合适的缩放比例。
